Dataset analysis

Variable Type Range Description
PROVINCIA Integer [28] Province (Madrid)
MUNICIPIO Integer [79] Municipality
ESTACION Integer Not unique Identifier of the monitoring station
MAGNITUD Integer [1 - 44] Component of air monitored
PUNTO_MUESTREO String Not unique Province, municipality, station, monitoring and sampling technique
AÑO Integer [2022] Year
MES Integer [01 - 12] Month
DXX Float [0 - 44] Value monitored
VXX String [V, N] Validity of the value obtained (V ⇒ valid, N ⇒ not valid)

Resource naming strategy

  • Domain: http://smartcity.airquality.es
  • Path: http://smartcity.airquality.es/airquality
  • Data: http://smartcity.airquality.es/airquality/resource
  • Ontology: http://smartcity.airquality.es/airquality/ontology
    - Classes: http://smartcity.airquality.es/airquality/ontology/<ontologyName>#<className>
    - Individuals: http://smartcity.airquality.es/airquality/resource/<className>/<identifier>
    - Properties: http://smartcity.airquality.es/airquality/ontology/<ontologyName>#<propertyName>
